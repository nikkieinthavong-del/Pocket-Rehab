# Game Assets Documentation

## Asset Organization

### Symbols (`assets/images/symbols/`)
Game symbols that appear on the 6x5 grid:

1. **symbol_sad_brick.png** - Sad brick/block with green slime
   - The distressed brick character with a worried expression
   - Features green toxic slime dripping

2. **symbol_fist.png** - Purple wrapped fist
   - Purple/pink colored fist with a rusty metal band/wrap
   - Blood or stains visible on the band

### UI Elements (`assets/images/ui/`)
User interface components:

1. **bonus_panel.png** - Scatter rewards panel
   - Shows three bonus levels:
     - 3 SCATTERS: 100x (Green)
     - 4 SCATTERS: 200x (Cyan)
     - DETOX SPINS: (500x) (Red)
   - Rusty industrial panel design with wires

### Effects (`assets/images/effects/`)
Visual effects for game mechanics:

1. **multiplier_x2.png** - 2x multiplier indicator
   - Green neon "x2" on cracked concrete/stone background
   - Glowing effect around the text

2. **cluster_effect.png** - Cluster connection visualization
   - Green glowing network/web pattern
   - Shows connections between symbols in a cluster win

## Required Additional Assets

To complete the game, you'll need to create or source:

### Additional Symbols
- At least 6-8 different symbols for variety
- Consider theme: toxic, rehab, dark humor
- Suggested: pills, syringes, toxic waste, warning signs, etc.

### UI Elements
- Spin button
- Bet adjustment buttons
- Balance display
- Win counter display
- Settings button
- Sound toggle
- Paytable button

### Sound Effects
- Spin sound
- Win sound
- Scatter landing sound
- Bonus trigger sound
- Background music

### Backgrounds
- Main game background
- Bonus round background
- Win celebration background

## Asset Specifications

- **Format**: PNG with transparency
- **Symbol Size**: 200x200px recommended
- **UI Elements**: Variable, maintain aspect ratio
- **Effects**: Variable, optimized for overlay
