# Asset Placement Instructions

Please place the provided game assets in the following locations:

## Images to Place:

1. **Taser weapon image** → `images/items/taser.png`
2. **Angry brick symbol** → `images/symbols/brick_angry.png`
3. **Post-apocalyptic cityscape** → `images/backgrounds/background_city.png`
4. **Dark corridor with fence** → `images/backgrounds/background_corridor.png`
5. **Betting monitors UI** → `images/ui/betting_monitors.png`

## Directory Structure:
```
assets/
├── images/
│   ├── items/
│   │   └── taser.png
│   ├── symbols/
│   │   └── brick_angry.png
│   ├── backgrounds/
│   │   ├── background_city.png
│   │   └── background_corridor.png
│   └── ui/
│       └── betting_monitors.png
└── sounds/
    └── (future sound effects)
```

Once assets are placed, the game will load them automatically.
