## Asset Requirements

All assets use a gritty, post-apocalyptic aesthetic with:
- Weathered/damaged textures
- Dark, moody color palette
- Glitch effects and distortion
- Industrial/urban decay theme

## Grid Layout
- 6x5 cluster slot configuration
- Symbols should be ~100-120px square for optimal display
- Background should support 16:9 or wider aspect ratios
